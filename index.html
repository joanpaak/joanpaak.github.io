Click on the names for short descriptions. Click on the arrows to go to the repository.

General Recognition Theory (GRT) is a multidimensional generalization of Signal Detection Theory. There have been some resources for Bayesian inference, but in my opinion they have been clunky to use (I mean no offence!).

Stan is a modern and easy-to-use tool for Bayesian inference. Here I present simple Stan programs for analyzing the ubiquitous 2X2 identification experiment.

>>>

Bayesian models are routinely approximated using random sampling. One possible way of doing this is by using the Resample-Move algorithm. Here's a Java implementation of it.
